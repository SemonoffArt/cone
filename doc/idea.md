# Графическое приложение для измерения объёма конуса

## Описание проекта

Приложение предназначено для расчёта объёма конических куч руды на основе анализа изображений. Пользователь загружает фотографию кучи, рисует треугольник, определяющий её профиль, и приложение автоматически вычисляет объём с использованием геометрических формул.


## Основная идея

Измерение объёма конических объектов через:
1. **Загрузку изображений** - локальные файлы или видеопоток с камер Trassir
2. **Рисование треугольника** - пользователь отмечает вершину и основание конуса на изображении
3. **Масштабирование** - установка реального размера пикселя (по умолчанию 0.1m)
4. **Расчёт объёма** - формула V = ⅓πR²h на основе геометрии треугольника
5. **Экспорт результатов** - копирование данных в буфер обмена


## Дальнейшее развитие

Возможные улучшения:
- История измерений с сохранением результатов
- Калибровка камер для более точных измерений
- Экспорт в различные форматы (CSV, PDF)
- Улучшенная визуализация 3D конуса
- Распознание конуса с помощью компьютерного зрения
