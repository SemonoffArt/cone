# Презентация программы «Cone App»
## Расчёт объёма и массы конуса руды на складе

---

## 1. Предпосылки создания программы

**Запрос от руководства:**
- Инициатива поступила от **директора фабрики** и **главного металлурга**
- Необходимость автоматизации расчёта объёма и массы руды в конусах складирования
- Использование существующей системы видеонаблютия для получения оперативных данных

**Текущая проблема:**
- Отсутствие инструментов для быстрого расчёта объёма руды по видеоаналитике
- Необходимость оперативного контроля запасов сырья на складах
- Потребность в точных данных для планирования производства

---

## 2. Описание решения

**Cone App** — специализированное приложение для расчёта объёма и массы конических складов руды на основе видеоизображений с камер видеонаблюдения Trassir.

### Ключевые возможности:

1. **Интеграция с системой Trassir**
   - Автоматическая загрузка снимков с камер наблюдения
   - Поддержка камер на объектах ЗИФ-1 и ЗИФ-2
   - Работа в реальном времени

2. **Автоматическое распознавание конуса**
   - Алгоритм компьютерного зрения для автоматического построения треугольника
   - Определение границ конуса на изображении
   - Возможность ручной корректировки при необходимости

3. **Точный расчёт параметров**
   - Объём конуса (м³)
   - Масса руды (тонны)
   - Радиус основания и высота конуса
   - Учёт коэффициентов объёма и плотности

4. **Удобный интерфейс**
   - Визуальное отображение треугольника на изображении
   - Панель с детальной информацией
   - Возможность масштабирования изображения
   - Быстрое копирование результатов в буфер обмена

## 3. Технические характеристики

### Архитектура:
- **Язык программирования:** Python 3.13+
- **Графический интерфейс:** Tkinter
- **Обработка изображений:** OpenCV, Pillow
- **Система видеонаблюдения:** Trassir

### Конфигурация камер:

**ЗИФ-1 (19. Конус Руда):**
- IP-адрес: 10.100.59.10
- Размер пикселя: 0.135 м
- Коэффициент объёма: настраиваемый
- Коэффициент плотности: 1.7 т/м³

**ККД-2 (115. Конус):**
- IP-адрес: 10.100.72.14
- Размер пикселя: 0.16 м
- Коэффициент объёма: настраиваемый
- Коэффициент плотности: 1.7 т/м³

## 4. Рабочий процесс

### Шаг 1: Загрузка изображения
- Автоматическая загрузка снимка с камеры при запуске программы
- Возможность переключения между камерами ЗИФ-1 и ЗИФ-2
- Поддержка загрузки локальных файлов изображений

### Шаг 2: Построение треугольника
- **Автоматический режим:** программа самостоятельно распознаёт контуры конуса
- **Ручной режим:** пользователь может построить/скорректировать треугольник вручную
- Перемещение вершин треугольника для точной подгонки

### Шаг 3: Получение результатов
- Мгновенный расчёт объёма и массы
- Отображение всех параметров конуса
- Копирование данных в Excel одним нажатием

---

## 5. Преимущества для производства

### Экономия времени:
- Расчёт занимает **менее 1 минуты** вместо ручных измерений
- Автоматическая интеграция с системой видеонаблюдения
- Не требует выезда на объект

### Точность данных:
- Математически точный расчёт по формуле V = ⅓πR²h
- Учёт калибровочных коэффициентов для каждой камеры
- Возможность ручной корректировки при необходимости

### Оперативность:
- Данные доступны в любой момент времени
- Быстрое переключение между объектами
- Мгновенный экспорт результатов

### Безопасность:
- Исключается необходимость доступа персонала к складу руды
- Снижение рисков травматизма
- Дистанционный мониторинг
